# 🪙 Coin Tosser with Tasks

A simple and interactive Coin Toss Simulator built using HTML, CSS, and JavaScript.

Click the coin to flip it — it spins and randomly lands on **Heads** or **Tails**, then assigns a task based on your input.

---

## 🚀 Features

- 🪙 Click-to-spin coin animation  
- 🎯 Random Heads or Tails result  
- ✍️ Custom task input for each outcome  
- ⚡ Instant result display  
- 🎨 Clean and modern UI  
- 📱 Responsive design  

---

## 🛠️ Technologies Used

- HTML5  
- CSS3  
- JavaScript  

---

## 📂 How to Run

1. Create a file named `index.html`  
2. Paste the project code into it  
3. Open the file in any web browser  

---

## 🎮 How to Use

1. Enter tasks:
   - Task for Heads  
   - Task for Tails  

2. Click the coin 🪙  

3. View:
   - Result (Heads/Tails)  
   - Assigned task  

---

## 💡 Example

- Heads → Study 1 hour  
- Tails → Watch a movie
